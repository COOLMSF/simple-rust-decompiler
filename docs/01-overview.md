## 1. 什么是反编译器

编译器把 C 代码变成机器码（二进制），反编译器做相反的事。

```
C 源码 ──[编译器]──> 二进制程序
二进制程序 ──[反编译器]──> C 代码
```

反编译出来的 C 代码不会和原始代码一模一样（变量名、注释都丢了），但逻辑是等价的。
